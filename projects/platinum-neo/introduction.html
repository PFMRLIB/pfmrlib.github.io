Project Platinum-Neo - 现代化SDR业余电台

注：由于项目仍在开发中，故下文参数可能与最终参数不同。
与此项目有关的文章见Tags:Platinum-Neo

特性

  • 支持多种模拟调制方式(AM, FM, SSB, CW)
  • 支持多种数字调制方式(RTTY, FT-8)
  • 可自行修改调制器的各项配置参数
  • 支持70MHz~6GHz内的所有业余频段
  • 电台存储记忆功能

简介

Project Platinum-Neo是一款全功能数字电台，源于此前的Project Platinum项目。此项目基于ADI的AD936x收发器实现，支持70MHz~6GHz范围内所有业余波段的操作，并且具有AGC、可调基带滤波器等必要功能。此设备支持多种模拟与数字调制方式，如AM、FM、SSB、RTTY等，并且调制的各项参数均可由用户调节。此外，此设备还具有自动呼叫、预制电文发射、CTCSS等多种辅助功能。

注:此项目的名称Platinum源于明日方舟干员白金。

架构

1. 硬件架构

Platinum-Neo的硬件架构如下所示：

Platinum-Neo主要由以下五部分构成：

  1. 射频收发模块。
    该部分包括AD936x宽带收发器以及用于射频收发的AFE。接收AFE由LNA、滤波器、VGA三部分组成，发射AFE由VGA、滤波器组、PA三部分组成。AFE通过自定义控制接口(包含一个SPI接口、一对模拟输入输出接口以及三个GPIO接口)连接到MCU与AD936x芯片的辅助AD/DA上。
    AD936x与FPGA间通过并行CMOS端口传输基带数据。MCU通过SPI与其他控制用GPIO端口控制AD936x的工作。
  2. FPGA。
    该部分基于20k级FPGA(XC7A25/35T, EP4CE22, Gowin GW2AR-18等)实现基带信号处理、语音处理、数据处理等功能。目前版本此部分采用了Sipeed的Tang Primer 20k板卡(GW2A-LV18)。FPGA通过I2S接口与音频CODEC间传输音频数据，通过FSMC接口与FPGA互传数据。
  3. MCU
    该部分基于STM32F407 MCU，控制整个系统的工作并通过USB FS接口与计算机通信。
  4. 音频CODEC
    音频CODEC将来自麦克风等输入设备的音频信号转化为输入数据流，或将输出数据流转化为音频信号从而驱动扬声器等输出设备。MCU通过I2C控制音频CODEC。
  5. 时钟源。
    系统采用了26MHz的TCXO(±0.5ppm)作为初始时钟源，并通过SI5351时钟发生器产生供给宽带收发器、FPGA与音频CODEC的时钟。MCU通过I2C控制时钟发生器。
